# Bank Application (Java + MySQL)

A console-based Bank Management System built using Java and MySQL with JDBC integration.  
This project allows users to create an account, login securely, and perform basic banking operations with persistent database storage.

---

## Features

- Create new bank account
- Secure login using Contact Number and PIN
- Deposit money
- Withdraw money
- Check account balance
- View transaction history
- Data stored permanently in MySQL database
- Uses PreparedStatement to prevent SQL injection
- External configuration for database credentials

---

## Tech Stack

- Java (Core Java)
- JDBC
- MySQL
- VS Code / IntelliJ
- MySQL Workbench
